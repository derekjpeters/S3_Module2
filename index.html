Coding Challenges

Challenge 1: Social Media Feature Implementation (Using forEach)

Scenario: You are tasked with implementing a feature for a social media platform that enables users to mark posts as "favorites". Your job is to efficiently count and display the total number of favorite posts.
Task: Develop a function that iterates over an array of post objects and increments a counter each time a post is marked as a favorite. Display the total count of favorite posts to the user.

Challenge 2: Project Data Preparation (Using forEach)

Scenario: As part of a project management tool, you need to format project names in a certain way for reporting purposes.
Task: Write a function that takes an array of project names and formats them according to the specified requirements (e.g., uppercase, prefixed/suffixed). Ensure the original data is not mutated.

Challenge 3: Inventory Management (Using filter)

Scenario: You are developing a feature for an e-commerce platform that requires separating available products from those that are out of stock based on inventory data.
Task: Implement a function that filters out products marked as "out of stock" from the product listings. The function should return a new array containing only the products available for sale.

Challenge 4: Financial Data Processing (Using map)

Scenario: In a financial application, you need to convert a list of transactions from one currency to another to display to the user.
Task: Create a function that applies a currency conversion rate to each transaction in an array. The function should return a new array with the converted values, leaving the original array unchanged.

Sprint Challenge: Implement sumOddInts

Objective: Write a function named sumOddInts that takes an array of numbers as input and returns a new array containing two elements: the sum of all odd integers, and an array of those odd integers.
Requirements: Use forEach to iterate over the array efficiently. Ensure the function handles edge cases, such as empty arrays or arrays without odd numbers, by returning [0, []].
Procedure: Initialize a sum variable and an array to hold odd numbers. Iterate over the input array, adding odd numbers to the sum and the odd numbers array. Return a new array with the sum and the array of odd numbers.
